## üíª Command Prompt (CMD) Basics

**The command line is powerful!** Here are essential commands for navigating and managing files.

### Navigation

| Command | What It Does | Example |
|---------|--------------|---------|
| `cd <folder>` | Change directory | `cd Documents` |
| `cd ..` | Go up one level | `cd ..` |
| `cd \` | Go to root of drive | `cd \` |
| `cd /d D:` | Change to D: drive | `cd /d D:` |
| `dir` | List directory contents | `dir` |
| `dir /a` | List all including hidden | `dir /a` |
| `cls` | Clear screen | `cls` |

### File Operations

| Command | What It Does | Example |
|---------|--------------|---------|
| `mkdir <name>` | Create folder | `mkdir Projects` |
| `rmdir <name>` | Remove empty folder | `rmdir OldFolder` |
| `rmdir /s <name>` | Remove folder and contents | `rmdir /s OldProject` |
| `copy <src> <dest>` | Copy file | `copy file.txt backup.txt` |
| `xcopy <src> <dest> /s` | Copy folder recursively | `xcopy folder newfolder /s` |
| `move <src> <dest>` | Move/rename file | `move old.txt new.txt` |
| `del <file>` | Delete file | `del unwanted.txt` |
| `del /s /q <folder>` | Delete all in folder quietly | `del /s /q temp` |
| `ren <old> <new>` | Rename file | `ren old.txt new.txt` |
| `type <file>` | Display file contents | `type readme.txt` |
| `echo <text> > file` | Write text to file | `echo Hello > note.txt` |
| `echo <text> >> file` | Append text to file | `echo More >> note.txt` |

### System Commands

| Command | What It Does |
|---------|--------------|
| `systeminfo` | Display system info |
| `hostname` | Show computer name |
| `whoami` | Show current user |
| `tasklist` | List running processes |
| `taskkill /pid <PID>` | Kill process by PID |
| `taskkill /im <name>` | Kill process by name |
| `ipconfig` | Show IP configuration |
| `ipconfig /all` | Detailed network info |
| `ipconfig /flushdns` | Clear DNS cache |
| `ping <host>` | Test connectivity |
| `tracert <host>` | Trace route to host |
| `netstat -an` | Show network connections |
| `shutdown /s /t 0` | Shutdown immediately |
| `shutdown /r /t 0` | Restart immediately |
| `sfc /scannow` | Scan and repair system files |
| `chkdsk C: /f` | Check disk for errors |

## üî∑ PowerShell Essentials

**PowerShell is CMD on steroids!** More powerful and modern, great for automation.

### Basic Commands

| Command | What It Does |
|---------|--------------|
| `Get-Location` or `pwd` | Current directory |
| `Set-Location` or `cd` | Change directory |
| `Get-ChildItem` or `ls` or `dir` | List contents |
| `New-Item -ItemType Directory` | Create folder |
| `New-Item -ItemType File` | Create file |
| `Copy-Item` or `cp` | Copy |
| `Move-Item` or `mv` | Move |
| `Remove-Item` or `rm` | Delete |
| `Get-Content` or `cat` | Read file |
| `Set-Content` | Write to file |
| `Add-Content` | Append to file |
| `Clear-Host` or `cls` | Clear screen |

### Useful PowerShell Commands

| Command | What It Does |
|---------|--------------|
| `Get-Process` | List all processes |
| `Stop-Process -Name <name>` | Kill process |
| `Get-Service` | List services |
| `Start-Service <name>` | Start service |
| `Stop-Service <name>` | Stop service |
| `Get-Command` | List all commands |
| `Get-Help <command>` | Get help for command |
| `Get-History` | Command history |
| `Invoke-WebRequest <url>` | Download from URL |

### PowerShell Tips

```powershell
# Find files by name
Get-ChildItem -Recurse -Filter "*.txt"

# Find files containing text
Select-String -Path "*.log" -Pattern "error"

# Get file sizes
Get-ChildItem | Select-Object Name, Length

# Sort by size
Get-ChildItem | Sort-Object Length -Descending

# Export to CSV
Get-Process | Export-Csv processes.csv
```
